# Weather API

Spring Boot REST API that retrieves current weather information for a city using the OpenWeather API.

## Features

- REST endpoint for current weather
- OpenWeather API integration
- Input validation
- Exception handling
- Swagger/OpenAPI documentation
- Response caching
- Unit and controller tests

## Technologies

- Java 17
- Spring Boot 3.5
- Maven
- Spring Web
- Spring Validation
- Spring Cache
- OpenAPI / Swagger

## Run application

Set environment variable:

```bash
OPENWEATHER_API_KEY=your_api_key
